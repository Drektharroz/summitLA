# Summit Legal Advisors - Landing Page

Sitio web moderno para presentar el estudio jurídico **Summit Legal Advisors** y sus áreas de práctica:

- Asesoría laboral
- Asesoría corporativa
- Asesoría tributaria
- Penal económico

## Funcionalidades destacadas

- Diseño responsive moderno con secciones de alto impacto.
- Modo oscuro con persistencia en `localStorage`.
- Contadores animados de indicadores clave.
- Evaluador interactivo de riesgo legal inicial.
- Slider de testimonios con controles.
- FAQ desplegable y animaciones de entrada al hacer scroll.
- Formulario de contacto con validación básica.

## Ejecutar localmente

```bash
python3 -m http.server 4173
```

Luego visita `http://localhost:4173`.
