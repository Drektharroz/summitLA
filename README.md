# Summit Legal Advisors - Landing Page

Sitio web estático moderno para presentar el estudio jurídico **Summit Legal Advisors** y sus áreas de práctica:

- Asesoría laboral
- Asesoría corporativa
- Asesoría tributaria
- Penal económico

## Ejecutar localmente

Basta abrir `index.html` en el navegador.

Si prefieres levantar un servidor local:

```bash
python3 -m http.server 4173
```

Luego visita `http://localhost:4173`.
